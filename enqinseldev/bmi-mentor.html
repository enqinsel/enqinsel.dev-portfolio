Project Overview

With this application, users can calculate their body mass index by entering their height and weight. According to the result, a diet list is given to the user.

Features

  • *When the page is first opened, the screen where the user's body mass index will be calculated by default is displayed. After the calculation is made, the BMI account and the diet list for the user are displayed on the screen.
  • *It consists of 6 components and 1 hook(onMounted) is used. Modal-specific SCSS is used.
  • *It is a project that can be developed. It can be called version 1 for now.
